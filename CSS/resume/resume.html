Border-Box

A opção "border-box" no box-sizing é especialmente útil quando se deseja especificar dimensões de elementos que incluam bordas e paddings. Isso pode ser útil, por exemplo, para garantir que um elemento tenha uma determinada largura ou altura, mesmo que tenha bordas ou paddings adicionais.

Box-Shadow

            Os valores que você pode usar para definir a sombra incluem:
            
                offset-x (necessário): valor para o deslocamento horizontal da sombra
                offset-y (necessário): valor para o deslocamento vertical da sombra
                blur-radius (opcional): valor para a distância de desfoque da sombra
                spread-radius (opcional): valor para a distância de espalhamento da sombra
                color (opcional): valor para a cor da sombra
             
            Exemplo:    
                
                .elemento {
                    box-shadow: 10px 5px 5px black;
                }
